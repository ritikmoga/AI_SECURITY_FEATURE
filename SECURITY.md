# Security Policy

This project is intended for defensive scanning and educational security analysis.

## Rules

- Do not upload live malware samples to issues or pull requests.
- Do not include real API keys, tokens, cookies, or credentials in commits.
- Do not use this project to attack third-party systems.
- Do not execute uploaded files during analysis.

## Reporting Issues

Open a GitHub issue with:

- A clear summary
- Steps to reproduce
- Expected behavior
- Safe sample data only

Do not include secrets or dangerous samples.
